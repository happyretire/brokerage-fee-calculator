# 2026 부동산 중개보수 & 취득세 계산기 (Real Estate Brokerage Fee & Tax Calculator)

🚀 **웹사이트 바로가기**: [https://happyretire.github.io/brokerage-fee-calculator/](https://happyretire.github.io/brokerage-fee-calculator/)

2026년(2025년 기준) 최신 법정 **중개보수 요율**과 **취득세 세율**을 동시에 확인할 수 있는 웹 애플리케이션입니다.
별도의 복잡한 설치 없이 브라우저에서 바로 사용할 수 있으며, 주택 수와 규제 지역 여부에 따른 복잡한 세금 계산도 간편하게 해결해줍니다.



## ✨ 주요 기능

*   **중개보수 계산**:
    *   최신 요율 자동 적용 (주택, 오피스텔, 상가 등)
    *   주거용 오피스텔(전용 85㎡ 이하) 자동 우대 세율 적용
    *   협의 요율 직접 입력 및 상한 체크
*   **취득세 예상액 계산** (New!):
    *   **다주택자 중과세 반영**: 1주택 ~ 4주택 이상 보유세율 자동 계산
    *   **규제 지역 옵션**: 조정대상지역 여부에 따른 세율 차이 반영
    *   **농어촌특별세**: 전용면적 85㎡ 초과 여부에 따른 농특세 추가 계산
*   **사용자 편의성**:
    *   🌙 **다크 모드 지원**: 눈이 편안한 테마 전환
    *   **실시간 계산**: 입력 즉시 결과 업데이트
    *   원단위 한글 표기 (예: 1억 5,000만 원)

## 🛠 사용 기술

*   **Icons**: [Lucide Icons](https://lucide.dev/)
*   **Compatibility**: 모든 모던 브라우저 (Chrome, Edge, Safari, Firefox 등)

## 📊 요율 및 세율 기준 (2026년)

### 1. 중개보수
*   **주택**: 거래 금액 구간별 상한 요율 (0.3% ~ 0.7%) 적용
*   **주거용 오피스텔**: 매매 0.5%, 임대차 0.4%
*   **그 외**: 0.9% 이내 협의

### 2. 취득세 (매매 시)
*   **1주택자**: 1~3% (가액별 차등)
*   **다주택자/법인**: 주택 수 및 조정지역 여부에 따라 8% 또는 12% 중과
*   **오피스텔/상가**: 4.6% (단일 세율)
